# ManagementAI

## 目的
介護会社の管理部業務を支援するAI・情報管理プラットフォーム。

## Ver1.0の対象機能

### 1. ニュース管理
- 労務ニュース取得
- 助成金・補助金情報取得
- 介護制度情報取得
- AI要約
- Chatwork配信
- 週報・月報

### 2. Webダッシュボード
- ニュース一覧
- 重要度別表示
- カテゴリ別表示
- キーワード検索
- 詳細画面

### 3. 管理部FAQ
- FAQ一覧
- FAQ検索
- ニュースからFAQ候補作成

### 4. AI分析
- 要約
- 会社への影響
- 管理部対応案
- 重要度判定
- 会社関連度判定

### 5. 最後に統合するもの
- 既存AI_Bot
- Chatwork質問応答
- FAQ検索回答
- ニュースDB参照回答

## 開発方針
- AI_Botとの統合は最後に行う
- まずはRoumu_News_Bot機能をManagementAI内のNews機能として完成させる
- Web画面はFlaskで作成する
- GitHubで管理する
- 本番投稿前は必ずAPP_MODE=testで確認する
